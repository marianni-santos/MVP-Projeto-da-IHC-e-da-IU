# GoldenPet - Projeto de IHC e UX

## 📖 Sobre o Projeto

Este repositório contém todos os artefatos desenvolvidos para o projeto **GoldenPet**, elaborado na disciplina de Interação Humano-Computador (IHC) e User Experience (UX).

O objetivo do projeto é projetar e validar uma solução digital para facilitar o gerenciamento da saúde e dos cuidados de animais de estimação, utilizando métodos e técnicas de UX durante todo o processo de desenvolvimento.
